# LearnEdge - AI-Powered Adaptive Learning Platform

LearnEdge is a production-grade full-stack platform that acts as an intelligent tutor. It extracts knowledge from your study materials (PDF, DOCX, TXT) and generates personalized learning experiences.

## Core Features

- **AI Content Extraction**: Upload materials and get instant topic analysis.
- **Dynamic Quiz Generation**: Mixed-mode quizzes (MCQ, Short, Long) generated by Google Gemini.
- **Intelligent Evaluation**: Answers are scored with detailed feedback, analogies, and memory tricks.
- **Progress Tracking**: Visualizes mastery levels per topic.
- **Premium UI**: Modern glassmorphism design with a focus on user experience.

## Tech Stack

- **Frontend**: Next.js 14, Tailwind CSS, Axios.
- **Backend**: NestJS, Prisma (SQLite), Passport/JWT.
- **AI**: Google Gemini Pro 1.5.

## Getting Started

1. **Clone the repository**
2. **Setup Environment Variables**:
   - In `server/.env`: Add your `GEMINI_API_KEY`.
3. **Install Dependencies**:
   ```bash
   npm run install:all
   ```
4. **Run Development Mode**:
   ```bash
   npm run dev
   ```

## Folder Structure

- `/client`: Next.js frontend.
- `/server`: NestJS backend.
- `/server/prisma`: Database schema and migrations.
